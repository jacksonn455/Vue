1) Instalando o CLI do Vue
npm install -g vue-cli@2.7.0

2) Novo projeto a partir de um template
vue init webpack-simple alurapic

3) baixar as dependencias
npm install

4) Quando todas as dependências forem baixadas,
já podemos subir nossa aplicação através do comando:
 npm run dev

5)Comunicação com web services
npm install vue-resource@1.0.3 --save

6) Carregamento de um ou outro componente de acordo com uma rota específica
npm install vue-router@2.1.1 --save

7)Estrutura de validação de formulário
npm install vee-validate@2.0.0-beta.18 --save
